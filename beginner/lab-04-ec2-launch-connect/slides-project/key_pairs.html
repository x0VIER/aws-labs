Step 2: Create and Manage Key Pairs

Securing access to your EC2 instances

Key pairs are used to securely connect to your EC2 instances. The public key is stored on the instance, and you use the private key to authenticate.

1

In the EC2 console, navigate to Network & Security > Key Pairs.

2

Click Create key pair.

3

Enter a Name for your key pair (e.g., my-ec2-key).

4

Select RSA for key pair type and .pem for private key format.

5

Click Create key pair to download the private key file.

Important: Store your private key file securely. If you lose it, you cannot recover it and will need to create a new key pair.

AWS CLI Alternative:

# Create a new key pair and save the private key to a file
aws ec2 create-key-pair \
  --key-name my-ec2-key \
  --query 'KeyMaterial' \
  --output text > my-ec2-key.pem

# Set proper permissions for the key file (Linux/macOS)
chmod 400 my-ec2-key.pem
