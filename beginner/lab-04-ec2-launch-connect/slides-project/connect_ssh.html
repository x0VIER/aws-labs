Step 3: Connect via SSH

Accessing your EC2 instance securely

SSH (Secure Shell) is a secure protocol for connecting to your EC2 instance. The connection process varies slightly depending on your operating system.

1

Locate your instance's Public IPv4 address in the EC2 console.

2

Ensure your key pair file (.pem) has the correct permissions.

3

Use the SSH command with your key file and the instance's public IP address.

4

Accept the connection when prompted (first connection only).

5

You should now be connected to your EC2 instance's command line.

Linux/macOS Windows
# Set correct permissions for your key file
chmod 400 my-key-pair.pem

# Connect to your instance
ssh -i my-key-pair.pem ec2-user@ec2-198-51-100-1.compute-1.amazonaws.com

# For Amazon Linux 2023, use 'ec2-user' as the username
# For Ubuntu AMIs, use 'ubuntu' as the username
# Using Windows PowerShell
ssh -i my-key-pair.pem ec2-user@ec2-198-51-100-1.compute-1.amazonaws.com

# Using PuTTY
# 1. Convert .pem to .ppk using PuTTYgen
# 2. Configure PuTTY with your .ppk file
# 3. Enter hostname: ec2-user@ec2-198-51-100-1.compute-1.amazonaws.com
