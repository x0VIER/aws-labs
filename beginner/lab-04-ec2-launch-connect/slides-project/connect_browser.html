Step 4: Connect via EC2 Instance Connect

Browser-based access to your instance

EC2 Instance Connect provides a simple and secure way to connect to your Linux instances using only your browser, without requiring a key pair.

1

In the EC2 console, select your running instance.

2

Click Connect at the top of the page.

3

Select the EC2 Instance Connect tab.

4

Keep the default user name (e.g., ec2-user for Amazon Linux).

5

Click Connect to open a terminal window in your browser.

Note: EC2 Instance Connect requires that your instance has a public IPv4 address and is in a public subnet with internet access.
