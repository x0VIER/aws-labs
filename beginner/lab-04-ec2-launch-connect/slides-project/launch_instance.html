Step 1: Launch an EC2 Instance

Creating your virtual server

1

Navigate to the EC2 service in the AWS Console.

2

Click Launch instance.

3

Enter a Name for your instance (e.g., my-first-ec2).

4

Select an Amazon Machine Image (AMI) - Amazon Linux 2023 is recommended for this lab.

5

Choose an Instance type - t2.micro is eligible for the free tier.

6

Create or select a Key pair for SSH access.

7

Configure Network settings - allow SSH traffic (port 22).

8

Click Launch instance.

AWS CLI Alternative:

aws ec2 run-instances \
  --image-id ami-0c7217cdde317cfec \
  --instance-type t2.micro \
  --key-name my-key-pair \
  --security-group-ids sg-903004f8 \
  --subnet-id subnet-6e7f829e \
  --tag-specifications 'ResourceType=instance,Tags=[{Key=Name,Value=my-first-ec2}]'
