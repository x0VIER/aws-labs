Step 4: Sign In as the IAM User

Using your new administrative user

Now that you've created an IAM user with administrative permissions, you should sign out of the root user account and sign in with your new IAM user.

1

In the top-right corner of the AWS Management Console, click on your account name and select Sign Out.

2

Use the unique console sign-in link from the .csv file you downloaded earlier. It will look like:

https://123456789012.signin.aws.amazon.com/console

3

Enter the IAM user name (aws-admin) and the password you set.

4

Verify that you're logged in as the IAM user by checking the user name in the top-right corner.

Tip: Bookmark your IAM user sign-in URL for easy access in the future.

Security Best Practice: You are now following AWS security best practices by using an IAM user for your day-to-day AWS activities instead of the root user.
