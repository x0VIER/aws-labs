Step 3: Create an Administrative IAM User

Following AWS security best practices

Creating a separate IAM user for administrative tasks is a fundamental security best practice in AWS.

1

Navigate to the IAM service in the AWS Console.

2

Click Users and then Add users.

3

Enter a username (e.g., aws-admin) and select AWS Management Console access.

4

Choose I want to create an IAM user and set a custom password.

5

On the permissions page, select Attach policies directly.

6

Search for and select the AdministratorAccess policy.

7

Review the details and click Create user.

8

Download the .csv file with the user credentials.

Security Note: Store the credentials securely. This file contains the console sign-in link and password for your administrative user.
