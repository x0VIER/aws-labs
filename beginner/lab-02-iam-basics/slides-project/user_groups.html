Step 1: Create IAM User Groups

Organizing users with common permissions

User groups allow you to manage permissions for multiple users at once.

1

Navigate to the IAM service in the AWS Console.

2

Click User groups and then Create group.

3

Enter a group name and select the appropriate policy.

S3-Developers Group

Attached Policy: AmazonS3FullAccess

EC2-Support Group

Attached Policy: AmazonEC2ReadOnlyAccess

AWS CLI Alternative:

# Create the S3-Developers group
aws iam create-group --group-name S3-Developers

# Attach the S3 full access policy
aws iam attach-group-policy \
  --group-name S3-Developers \
  --policy-arn arn:aws:iam::aws:policy/AmazonS3FullAccess
