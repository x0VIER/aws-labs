Step 3: Create IAM Users and Assign to Groups

Creating identities and granting permissions

1

In the IAM console, click Users and then Add users.

2

Enter a username (e.g., s3-dev-1) and select console access.

3

Set a password and click Next.

4

Select Add user to group and choose the appropriate group.

5

Review and click Create user.

AWS CLI Alternative:

# Create the s3-dev-1 user
aws iam create-user --user-name s3-dev-1

# Add user to the S3-Developers group
aws iam add-user-to-group \
  --user-name s3-dev-1 \
  --group-name S3-Developers
