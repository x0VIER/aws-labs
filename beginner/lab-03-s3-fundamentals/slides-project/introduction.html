Introduction to Amazon S3

Simple Storage Service

Amazon Simple Storage Service (S3) is an object storage service that offers industry-leading scalability, data availability, security, and performance.

Key features of Amazon S3:

Durability: 99.999999999% (11 9's) designed to protect against data loss
Scalability: Store virtually unlimited amounts of data
Storage Classes: Different tiers for different access patterns
Security: Multiple mechanisms to control access to your data

S3 is commonly used for backup and restore, data archiving, content distribution, data lakes, and static website hosting.
