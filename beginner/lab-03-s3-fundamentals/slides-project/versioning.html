Step 3: Enable Bucket Versioning

Protecting against accidental deletions and modifications

S3 versioning keeps multiple variants of an object in the same bucket, allowing you to preserve, retrieve, and restore every version of every object.

1

Navigate to your bucket in the S3 console.

2

Click the Properties tab.

3

Find the Bucket Versioning section and click Edit.

4

Select Enable and click Save changes.

5

Upload a new version of an existing file to test versioning.

6

Click Show versions to view all versions of your objects.

Note: Once enabled, versioning cannot be disabled, only suspended.

AWS CLI Alternative:

# Enable versioning on a bucket
aws s3api put-bucket-versioning \
  --bucket your-name-test-bucket-123 \
  --versioning-configuration Status=Enabled

# List all versions of objects in a bucket
aws s3api list-object-versions \
  --bucket your-name-test-bucket-123

* Additional storage costs apply for multiple versions
