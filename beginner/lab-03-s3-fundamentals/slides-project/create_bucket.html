Step 1: Create an S3 Bucket

Setting up your storage container

A bucket is a container for objects stored in S3. Every object is contained in a bucket, and bucket names must be globally unique.

1

Navigate to the S3 service in the AWS Console.

2

Click Create bucket.

3

Enter a globally unique Bucket name (e.g., your-name-test-bucket-123).

4

Select an AWS Region for your bucket.

5

Keep default Block Public Access settings and click Create bucket.

Bucket Naming Rules:

  • Between 3 and 63 characters long
  • Can contain only lowercase letters, numbers, dots, and hyphens
  • Must begin and end with a letter or number
  • Must be globally unique across all AWS accounts

AWS CLI Alternative:

aws s3api create-bucket \
  --bucket your-name-test-bucket-123 \
  --region us-east-1
