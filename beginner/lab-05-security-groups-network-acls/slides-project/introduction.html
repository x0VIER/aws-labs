Introduction to AWS Network Security

Defense in Depth with Multiple Security Layers

AWS provides multiple layers of network security to protect your resources. Two key components are Security Groups and Network Access Control Lists (NACLs).

Security Groups: Instance-level firewalls that control inbound and outbound traffic for EC2 instances
Network ACLs: Subnet-level firewalls that control inbound and outbound traffic for VPC subnets

Together, these components create a defense in depth strategy:

  • Network ACLs provide the first layer of defense at the subnet level
  • Security Groups provide the second layer of defense at the instance level
  • Both can be used together to create a comprehensive security posture
